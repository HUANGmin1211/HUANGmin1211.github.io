Java基础第八章，包

Posted on 2019-01-15 | In Java

包机制

为什么需要包？对于多个类为了便于管理（存在类的同名情况），Java中提供了包机制解决方案。落实到操作系统上，就是对Java的文件进行份文件夹处理

作用：对类文件进行管理；给类文件提供了名称空间（路径）

定义：使用关键字 package 包名; 包名中的所有字母都小写。包语句写在所有程序的第一行

总结

  • 包与包之间的类在访问时，被访问的类以及成员都必须public修饰
  • 被public修饰的类或者接口，所属的java文件名必须和类名或者接口名一只
  • 包与包之间继承，父类可以给其他包中的额子类提供一个特殊的权限 protected，只有继承为子类后，就可以访问的权限

权限总结


包与包之间访问只有两种权限可以用，public protected（该权限只能给不同包中的子类使用）

import

引入：【包名.子包名.类名】的出现导致类的名称过长，书写不方便。可以通过import解决
使用：import 包名.类名，将包中的类导入进来；import 包名.，将包中的所有的类导入进来
import导入的是包中的类，不会导入包中的包里面的类
作用：简化类名书写，省略包名
特殊情况
packa\packaa\DemoAA.class，packa\DemoA.class。那么使用import packa.，只会导入packa中的类，不会导入其子包packaa中的类。如果要使用DemoAA，需要import packa.packaa.*。
不同包中存在相同名称的类，使用该类时必须使用包名

jar包

Java中的压缩包

方便项目的携带，使用时可以直接将jar导入到classpath路径中即可
jar包里面是class文件，使用者需要配合API使用，我也可以把我自己写的类打包成jar包给别人使用，当然也要配合API
java是开源指的是开放源代码，他给你提供src.rar压缩包，但是这里面都是源代码文件，即.java文件
操作

创建jar包：jar  -cvf  mypack.jar  packa packb
查看jar包：jar  -tvf  mypack.jar   [>定向文件]
解压缩：jar  -xvf  mypack.jar
自定义jar包的清单文件：jar –cvfm  mypack.jar  mf.txt  packa packb
